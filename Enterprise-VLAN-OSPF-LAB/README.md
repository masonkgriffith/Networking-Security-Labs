# Enterprise VLAN & OSPF Network Lab

## Objective
Design and implement a segmented enterprise network using VLANs and dynamic routing.

## Network Design
- VLAN 10 – Users
- VLAN 20 – Servers
- VLAN 30 – Management
- VLAN 99 – Native

## Key Configurations
- Inter-VLAN routing via Layer 3 switch
- OSPF dynamic routing configuration
- DHCP pool assignment
- Routing table verification

## Verification & Testing
- Inter-VLAN ping testing
- OSPF neighbor validation
- Route table inspection
- Failover and convergence testing

## Skills Demonstrated
- Network segmentation
- Broadcast domain reduction
- Dynamic routing configuration
- Troubleshooting connectivity issues
