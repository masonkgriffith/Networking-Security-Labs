# Networking & Security Lab Portfolio

This repository contains hands-on enterprise networking and security labs designed to simulate real-world infrastructure environments.  

Each lab focuses on secure architecture design, segmentation, routing, firewall policy enforcement, and structured troubleshooting.

The objective of this portfolio is to demonstrate practical engineering ability beyond certification theory by building, validating, and documenting working infrastructure scenarios.
